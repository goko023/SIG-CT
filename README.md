INSTALLATION
------------
1. Extract all files
2. Run SIG_Generator.exe
  That's it.

USAGE
-----
  - Click "New Project" to create a new project
  - Add / select / delete items from the left panel
  - Edit item properties in the editor section
  - Live YAML preview on the right panel
  - Export with "YAML Export" to a .yml file
  - Save and load projects (.sigproj format)

PLUGIN
------
  SimpleItemGenerator by balerii
  https://www.spigotmc.org/resources/simpleitemgenerator.119378/

NOTES
-----
  - .sigproj files are JSON format and can be manually edited
